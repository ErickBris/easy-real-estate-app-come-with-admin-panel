This is a real estate application template run under Android platform and come with server build on php framework included. This will help you build your own mobile marketplaces application faster. With many great features.

  1. Admob Support with google play services
  2. Nice User Interfaces design
  3. User can login, edit profile, submit properties, edit properties
  4. Admin can manager all in backend panel
  5. Backend panel install easy by installer (input well done, no hand setup)
  6. The nearest properties around you (using gps)
  7. Filter Location with more optional
  8. Sliding Menu
  9. Custom marker
  10. Google maps intergration
  11. User guide in documentation file and video

  1. Download and unzip the package
  2. Upload the all file in cms folder to your public_html server
  3. Open .htaccess file


    4. Make sure your .htaccess file like this

    RewriteEngine On
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteRule ^(.*)$ /index.php/$1 [L]


  4. Using your browser navigate to yourdomain/install/index.php.
  5. Create an empty database in your mysql server
  6. Input the details specific to your setup using your empty database then submit

  7. After in install. Now the install wizard will redirect you to login dashboard, using your account when setup in installing to login.
  8. For security reason, please delete install folder after install.

  1. Changing admob publisher id

    change value of admob_publisher_id tag with your admob id

  2. Changing fanpage link

    change value of fan_page_linktag with your facebook fanpage link

  3. Config app to connect to server

    To connect to server, you only need change value of:

    +, domain_url tag to your domain where you upload php cms .

    +, type_json_url tag to http://your_domain/api/type_api.

    +, county_json_url tag to http://your_domain/api/county_api.

    +, estates_json_url tag to http://your_domain/api/estate_api.

    +, images_json_url tag to http://your_domain/api/images_api.

    +, cities_json_url tag to http://your_domain/api/cities_api.

    +, marker_json_url tag to http://your_domain/api/marker_api.

    +, amenities_json_url tag to http://your_domain/api/amenities_api.

    +, cities_json_url tag to http://your_domain/api/cities_api.

  1. Open eclipse, Select File - > Import

  2. Select Existing Android Code Into Workspace

  3. Navigate to where you extract code and select DroidMarket folder, after that, click OK

  4. Check Copy projects into workspaces, Select File - > Import, after that, click Finish

  5. Wait for eclipse import and build project, after that, if you see some folder have red error, don't worrry , go to Project -> Build Automaticall , after that, restart your eclipse. After restart eclipse,all done :)
